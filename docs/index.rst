CARROT
==========================

CARROT (CARe ROuting Tool) is a schedule and route planner designed for use in the Home Health Care (HHC) sector. CARROT produces fast, feasible solutions for daily schedules and allows the user to modify the desired objective for a tailored result.

This documentation provides information on the background of CARROT and how to utilize the program and its optional features.
